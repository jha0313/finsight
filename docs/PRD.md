# PRD: finsight

## 목표
CSV(카드 명세서·거래내역)를 업로드하면 Claude가 분석해 **지출 구조·이상 거래·절약 인사이트**를 대시보드로 보여주는 핀테크 SaaS.

## 사용자
- **첫 방문자**: 가치·신뢰(보안)·가격을 보고 가입 결정
- **신규 가입자**: 첫 명세서를 올려 빠른 인사이트(Aha)
- **Free 사용자**: 매달 지출을 가볍게 점검
- **Pro 구독자**: 이상 거래·절약 인사이트로 행동
- **재방문자(월 주기)**: 새 명세서 추가

## 핵심 기능
### Free (규칙·통계 기반, LLM 비용 0)
1. 카테고리별 지출 분류
2. 기간별 지출 추이
3. 구독 누수·이상 거래 탐지(룰베이스: 반복결제 패턴·이상치)

### Pro (생성형 AI)
4. AI 자연어 요약·절약 인사이트(Claude `claude-opus-4-8`)

- **게이팅**: 서버 DB 구독상태로만 판정. 미구독/쿼터소진 시에도 `/api/analyze`는 **200 + Free 결과 + Pro 잠금 상태**(`pro.status=locked|unavailable`)를 반환.

## 사용자 여정
랜딩 → 구글 로그인 → CSV 업로드 → Free 분석 → Pro 잠금 미리보기 → Polar 결제 → Pro 잠금해제 → 월 주기 재방문.

## MVP 제외 (Post-MVP)
- 다중 명세서 누적 추세 / 데이터 내보내기
- 수동 컬럼 매핑 UI (MVP는 대표 포맷 + 표준 컬럼 가정)
- 다통화 환산 (KRW 외 거래는 분리 표시)
- 관측성 지표 파이프라인 / reconciliation cron / rate-limit 인프라(Upstash)
- 계정삭제 UI (cascade FK는 유지)

## 디자인
- 한국어 우선
- 금융 데이터라 **신뢰감 있는 미니멀 UI** + 명확한 보안 메시지(랜딩)
- 차트 중심 대시보드 (카테고리 도넛 + 기간 추이 라인 + 이상거래 하이라이트)
