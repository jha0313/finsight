목적함수

채택/폐기의 단일 기준

objective.mjsmedian-of-3 · 회귀 가드 ε=1pt
// cell = page P × device D × category C
score(P,D,C) = median( run₁, run₂, run₃ )
objective = Σ w_C · score(P,D,C) / Ncells

KEEP  ⇐ objective ↑  AND  ∀cell: score ≥ baseline − ε
REVERT ⇐ 그 외 전부  // 한 카테고리라도 1pt 넘게 떨어지면 폐기

Lighthouse 점수는 실행마다 출렁이므로 타깃별 median-of-3로 잡음을 줄인다. 단일 카테고리라도 baseline 대비 1점 넘게 후퇴하면 그 변경은 폐기 — 한 점수를 올리려 다른 점수를 깎는 것을 막는다.
